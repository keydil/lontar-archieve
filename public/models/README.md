# /public/models/

Place your `.glb` 3D model files here.

Each model file should be named to match the artifact `slug` from `/src/data/koleksi.ts`.

Example:
- `tabungan-demo.glb` → matches slug `tabungan-demo`
- `daun-lontar-01.glb` → matches slug `daun-lontar-01`

If a `.glb` file is not found, the viewer will display a placeholder wireframe geometry.
